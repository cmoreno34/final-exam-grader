    How grading works

    For Excel questions the app unzips the .xlsx, parses the inner XML to extract every cell, formula, number-format, data-validation rule, conditional formatting rule, chart, pivot table, slicer, defined name, and Power Query connection. For each rubric question it sends Claude only the relevant slice of that data plus the rubric text and your strictness preset, then asks for a JSON verdict.

    For SQL screenshots the app sends the images to Claude with vision, plus the source CSV tables (so Claude can verify each query's expected result), and asks for one JSON verdict per question.

    All grading happens client-side. Your students' files never touch any server other than Anthropic's. Costs are billed to the API key you provide.
